---
title: Does Vendasta allow login via Username and Password on WordPress Hosting SFTP?
description: Information about login methods for WordPress Hosting SFTP access
sidebar_label: WordPress Hosting SFTP Login
---

No, at the moment only key files and usernames are allowed over SFTP, we are unable to provide SFTP passwords to Partners.

For more details, please refer to [How to use SFTP to access file system](/vendasta-products/vendasta-products-website/how-to-use-sftp-to-access-file-system).
