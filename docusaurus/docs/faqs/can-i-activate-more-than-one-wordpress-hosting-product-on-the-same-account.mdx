---
title: Can I activate more than one WordPress Hosting product on the same account?
sidebar_label: Multiple WordPress Hosting products
description: Learn about activating multiple WordPress Hosting products on the same account
---

Unfortunately, you cannot activate multiple units of the WordPress Hosting and Standard products on a single account. If a client has two websites they'd like to host with Vendasta, we recommend creating an additional account and activating WordPress Hosting on each one.
